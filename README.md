# Desafio-Estagio-Idealit

Desafio de admissão para o estágio de Desenvolvedor Multiplataforma na Idealit.

O projeto consiste de uma aplicação que atenda a seguinte requisição:

> Eu como usuário quero acessar o app com meu login e senha ao qual posso realizar o mesmo via e-mail ou facebook. <br> Quero antes de sair de casa para o supermercado, consultar a previsão do tempo pesquisando pelo CEP da localidade que pretendo ir, quero poder criar minha lista de compras e marcar o que foi comprado. Gostaria de verificar o poder do Real Brasileiro em relação a moedas de pelo menos 3 países simultaneamente. <br> Quero poder excluir minha conta na aplicação quando achar necessário.

Neste projeto será usado o [**Ionic**](https://ionicframework.com/) usando [**ReactJS**](https://pt-br.reactjs.org/) para desenvolver a aplicação e a [**Firebase**](https://firebase.google.com/) para realizar o deploy.
